# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2027.x   | ✅ |
| 2026.x   | :white_check_mark: |
| <2025.x   | :x:                |

## Reporting a Vulnerability

NomNom accepts security reports via GitHub issues. Please report them and label them with the `security` label.
